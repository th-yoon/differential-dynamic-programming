# differential-dynamic-programming

## Systems
- cartpole
- inverted pendulum
- 2D Quadcopter (in progress)
- 3D Quadcopter (in progress)

## Features
- partial derivatives using `jax`
- simulation on open ai gym, meta ai mbrl environments

## Requirements
- python 3.10
- pip install -r requirements.txt
